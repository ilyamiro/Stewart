## About the project
Stewart is a very useful voice assistant with a voice 
synthesizer provided by <a href="https://github.com/snakers4/silero-models">silero(v4)</a>
and a speech-recognition vosk models taken from <a href="https://alphacephei.com/vosk/models">there</a> <br>

Stewart uses GPT 3.5 models for question answering. <br>


## Getting Started
For deployment, you have to install python 3.11.6 from <a href="https://www.python.org/downloads/release/python-3116/">python.org</a> or use package manager on linux system.

### Installation
<strong>WARNING: the project is not finished</strong>
1. Clone repository:
  ```
  git clone https://github.com/ilyamiro/Stewart.git
  ```
2. Enter it:
  ```
  cd Stewart
  ```
3. Install requirements txt:
  ```
  python3.11 -m pip install -r requirements.txt
  ```
4. Launch gui.py using python: <br>
- Linux:
 
```
python3.11 GUI/console_gui.py
  ```
- On Windows:
